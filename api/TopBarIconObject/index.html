TopBarIconObject

The parent of all classes.

Properties​

Image​

</>
TopBarIconObject.Image: string | number

The image to display on the icon.

Name​

</>
TopBarIconObject.Name: string

The name of the icon.

Order​

</>
TopBarIconObject.Order: number

The layout order of the icon.

Area​

</>
TopBarIconObject.Area: number

The area of the icon. 1 is Left and 2 is Right.

StateChanged​

Event
</>
TopBarIconObject.StateChanged: SignalController<string>

Fires whenever the state of the icon changes. For example: Hovering -> Default

Activated​

Event
</>
TopBarIconObject.Activated: SignalController<Enum.UserInputType>

Fires when the icon is activated, by any supported input type. Also passes in the user input type enum.

NoticeAdded​

Event
</>
TopBarIconObject.NoticeAdded: SignalController<number>

Fires whenever a notice is added to the icon.

Notices​

</>
TopBarIconObject.Notices: number

The amount of notices the icon actually has.

NoticeCap​

</>
TopBarIconObject.NoticeCap: string

The notice cap string, an example of this is "99+" for the default.

CurrentState​

</>
TopBarIconObject.CurrentState: string

Describes the current state of the icon, works like how TopBarIconObject.StateChanged does.

Functions​

SetIconEnabled​

</>
TopBarIconObject:SetIconEnabled(
enabled: boolean--

Whether or not to enable the icon

) → ()

Sets the status of the icon visibility.

SetIconNotices​

</>
TopBarIconObject:SetIconNotices(
notices: number?,--

The amount of notices to add, leaving this nil will add a single notice

noticeCap: number?--

When to display a + sign after a set amount of notices, defaults to 99

) → ()

Adds notices to the parent topbar icon.

BindKeyCodes​

</>
TopBarIconObject:BindKeyCodes(
keyCodes: {Enum.KeyCode}?--

The key codes to listen to, if it is nil it will reset to no key codes

) → ()

Binds multiple key codes to activate the icons Activated event.

SetImageSize​

</>
TopBarIconObject:SetImageSize(
imageSize: Vector2--

The size to set the image to

) → ()

Sets the image size of the icon, default is filled.

Destroy​

</>
TopBarIconObject:Destroy() → ()

Destroys the icon itself, removing it from the topbar.
